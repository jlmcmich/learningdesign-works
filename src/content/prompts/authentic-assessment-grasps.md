Generates performance-task ideas aligned with learning objectives using the **GRASPS** framework.

## GRASPS Components

- **G**oal — Problem or goal definition.
- **R**ole — The student's role.
- **A**udience — Target audience.
- **S**ituation — Context or scenario.
- **P**roduct — What is created and why.
- **S**tandards — Rubrics or success criteria.

## Process

1. Request learning objectives.
2. Outline three performance tasks.
3. Explain alignment.
4. Solicit feedback for refinement.
