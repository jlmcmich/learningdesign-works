Creates clear, concise assignment descriptions using the **Transparency in Learning and Teaching (TILT)** framework.

## TILT Components

- **Purpose** — Why students complete the assignment; connections to learning objectives.
- **Task** — What students must do; concrete steps and guidelines.
- **Criteria for Success** — How work is evaluated; success indicators and checklists.

## Four-Phase Process

1. Initiate consultation; gather assignment draft and learning objectives.
2. Consider Purpose, Task, and Criteria strategy.
3. Draft TILT-structured assignment (verbose first draft).
4. Provide instructor-facing transparency rationale and refine.

## Output Format

- Student-Facing Assignment Description
- Instructor-Facing Transparency Rationale & Next Steps
- Optional Canvas-Friendly HTML Version
