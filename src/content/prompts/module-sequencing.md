Helps organize course content into modules aligned with course learning objectives using the **Understanding by Design** framework.

## Three-Phase Process

### Phase 1
Gathers course name, description, CLOs, and a comprehensive content list.

### Phase 2
Requests optional context (major assignments, existing structures, known challenges).

### Phase 3
Proposes a high-level course structure with modules, then adds MLOs and instructional approaches.

## Output

- **Design Rationale** — A narrative explanation of sequencing logic.
- **Course Sequence Blueprint** — For each module: position, focus, key content, CLOs addressed, MLOs, suggested learning approaches.
- **Connections** to other suite tools.
