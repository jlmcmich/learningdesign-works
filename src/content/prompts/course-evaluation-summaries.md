Provides structured qualitative trend analysis for student course-evaluation data.

## Analysis Sections

1. **What did you like most?** — themes with counts and direct quotes.
2. **What did you like least?** — themes with counts and direct quotes.
3. **Comments** — themes with counts and direct quotes.
4. **What should future students know?** — themes with counts and direct quotes.
5. **Sentiment analysis** overview.

## Output

A Course Evaluation Report with themes, frequencies, supporting quotes, and improvement recommendations.
